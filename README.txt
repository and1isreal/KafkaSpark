Для запуска приложения необходимо прописать в командной строке из classpath "mvn spring-boot:run".

Чтобы программа могла получить доступ к устройствам, потребуется запустить вышеуказанную команду от имени администратора.

Чтобы создать исполняемый JAR файл, необходимо набрать в коммандной строке из classpath "mvn clean compile package".
И запустить полученный архив командой "java -jar 'путь к архиву'"  (от имени администратора).

О программе:
- Таблица создается и заполняется значениями при запуске приложения во встроенной БД H2 database.
- Класс "Limit" представляет собой объектно-реляционную модель для таблицы "limits per hour".
- Pcap4j ловит все пакеты поступающие на устройство eth, и передает размер каждого пакета вв Spark Streaming.
- Spark Streaming высчитывает общий трафик за указанное время (5 мин) и отправляет сообщение в виде JSON объекта класса Limit.
- Kafka получает сообщения от Spark, а сервис UpdateLimitsService в свою очереь обновляет данные в таблице БД каждые 20 мин.
